# A Money Plan

A buffer-first avalanche debt payoff calculator and 18-month financial forecast tool.

## Features

- Buffer-first strategy (build emergency fund before aggressive debt payoff)
- Debt avalanche methodology (highest interest first)
- 18-month financial planning with customizable income and expenses
- Real-time calculations with local storage persistence
- Print-friendly dark theme
- CSV export functionality

## Live Demo

Visit the live application at: https://shabalalawatp.github.io

## Usage

1. Adjust your monthly income and expenses using the controls
2. View your buffer progress and debt payoff timeline
3. Monitor month-by-month financial projections
4. Export your plan to CSV for external analysis

## Local Development

Simply open `index.html` in a web browser. No build process or external dependencies required.
